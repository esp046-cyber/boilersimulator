# boilersimulator
Features included:
- Boiler Visualization - Animated tank with water level, steam, flames, and pump indicator
- Feedwater Pump Control - Start/stop with visual feedback
- Burner Sequence - 5-step sequence (Pre-purge → Pilot → Prove → Main Flame → Running)
- Limit Switches & Permissives - LWCO, High Pressure, Flame Detection, Purge status
- Level Control - Automatic pump logic based on setpoint
- Temperature/Pressure PID - Configurable Kp, Ki, Kd parameters
- Editable Variables - Setpoints, PID tuning, safety limits
- Ladder Logic Display - Real-time ladder diagram showing control logic states
- Trend Graph - Live chart tracking temp, pressure, and level

Navigation tabs:
- Main - Visualization, gauges, controls, permissives
- Logic - Ladder diagram and burner sequence status
- Vars - All editable setpoints and parameters
- Trend - Real-time process trends
